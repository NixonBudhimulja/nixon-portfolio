Category: Web Development

Pendekinlink.id

This is my best project, involving a collaborative effort to create a software solution. We developed a website where users can shorten URLs and enjoy various unique features, such as customizing shortened links, editing them (in progress), viewing history (in progress), and user login/signup. My role was to design the UI/UX in Figma and build the front-end using HTML and Tailwind CSS. The main challenge was ensuring a user-friendly UI/UX, but thanks to our collaborative teamwork, the design process went smoothly. The project is live, and you can check it out at pendekinlink.id.

The website is continuously being improved to offer even more advanced features, ensuring it meets user needs for simplicity and efficiency. As a collaborative project, we focused on making the user interface intuitive, while still providing a robust backend system to handle high volumes of shortened links. This project was an excellent opportunity for me to hone my front-end development skills and collaborate closely with other developers. We used modern design principles to ensure that the user experience was seamless across different devices, and we aim to continue expanding the features as we gather user feedback.


Sea of Ftars

This is my second best project, created as the final assignment for my Human-Computer Interaction course. The project involves developing a website for a PC game called Sea of Ftars, inspired by the original Sea of Stars game. The website includes pages like home, characters, fan art, and account login/register. I served as a full-stack developer, designing the UI/UX in Figma and building the site using HTML, CSS, and JavaScript in VS Code. Despite the challenges, I successfully completed the project and am very satisfied with the result. This was made possible through collaborative efforts with my friends, even though it was a solo project, as we often helped each other.
